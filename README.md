# Pi-Radio - Real time beamformer

This is built on top of Pi-Radio 8ch-if design.

## Repo Structure
The repo contains PL side vivado project, petalinux project and host side drivers to communicate with the RFSoC.
```
├── apu

│   ├── config
│   ├── demos
│   ├── helper

│   └── README.md
├── LICENSE
├── pl
│   ├── constrs
│   ├── project
│   ├── README.md
│   ├── scripts
│   └── srcs

```
